### unsloth_FineTuning_Llama-3.2-3B-instruct_custom_dataset
🚀 OverviewThe project demonstrates how to take a base instruct model and further refine it using a custom dataset. By leveraging Unsloth, you can achieve up to 2x faster training and 70% less memory usage compared to standard Hugging Face implementations.Key FeaturesModel: Llama-3.2-3B-Instruct.Optimization: Utilizes Unsloth for 4-bit quantization and efficient memory management.Format: Provided as an easy-to-follow Jupyter Notebook.License: Distributed under the MIT License.🛠️ Getting StartedPrerequisitesTo run the fine-tuning script, you will need:A GPU (NVIDIA T4, L4, or A100 recommended).Python 3.10+.Installed libraries: unsloth, xformers, bitsandbytes, and trl.InstallationYou can install the necessary dependencies directly within the notebook environment:Bashpip install "unsloth[colab-new] @ git+https://github.com/unslothai/unsloth.git"
pip install --no-deps "xformers<0.0.27" "trl<0.9.0" peft accelerate bitsandbytes
📖 UsageOpen the Notebook: Navigate to FineTunning_with_llama_unsloth_(1).ipynb.Load the Model: The script initializes the Llama-3.2-3B model in 4-bit precision to save VRAM.Prepare Dataset: Load your custom dataset in a format compatible with the Instruct template (System, Instruction, Input, Output).Train: Run the training cells to begin the LoRA (Low-Rank Adaptation) fine-tuning process.Inference: Test the fine-tuned model directly within the notebook to verify results.📂 Repository StructureFileDescriptionFineTunning_with_llama_unsloth_(1).ipynbThe main execution script for training.README.mdGeneral project documentation.LICENSEMIT License details.🤝 ContributingContributions are welcome!
